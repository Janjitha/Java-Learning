# Java-Learning 🚀  

A beginner-friendly collection of Java problems to practice and improve coding skills.  

## 📚 Topics Covered  
✅ Input & Output 
✅ Conditional Statements 
✅ Loops 
✅ Functions 
✅ Problem-Solving  

## 💡 How to Start?  
1. Browse the repository and pick a problem.  
2. Open the file, read the problem, and solve it in any Java compiler.  
3. Keep practicing and enhance your Java skills!  

## 🤝 Contribute & Learn Together!  
Add Java problems/solutions and submit a **Pull Request**. Let's grow together! 🚀
